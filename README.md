# Lista de tarefas

Projeto feito em Reactjs de uma lista de tarefas com consumo de API pelo axios, uso do toastify para gerar um alerta dinâmico informando que o usuário não pode add uma task com input vazio, além da opção da opção dele digitar a tarefa e apertar a tecla enter sem necessidade de clicar no botão.

## Veja o projeto completo:

https://the-great-keitey-site.netlify.app/
